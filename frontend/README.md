# Frontend (Vercel)

This folder contains the static frontend for the portfolio. Deploy this folder to Vercel.

Before deploying, set the `BACKEND_URL` in Vercel's environment variables (or edit the `backendUrl` variable) to point to your Render backend URL.

To preview locally:

```bash
# Serve from the frontend folder (example using Python)
cd frontend
python -m http.server 3000
# Then open http://localhost:3000
```
